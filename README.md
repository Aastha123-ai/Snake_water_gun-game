# Snake_water_gun-game



## Overview

A console-based Snake Water Gun game built using Python. The player competes against the computer, which randomly selects Snake, Water, or Gun.

## Rules

* Snake drinks Water → Snake Wins
* Water douses Gun → Water Wins
* Gun kills Snake → Gun Wins

## Features

* Random computer choice
* User input handling
* Win/Loss/Draw detection
* Simple and interactive gameplay

## Technologies Used

* Python
* Random Module

## Concepts Practiced

* Dictionaries
* Conditional Statements
* User Input Validation
* Random Number Generation
* Game Logic Design
